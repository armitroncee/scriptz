About This App

This is a full stack web-based app that allows people to keep track of their medications and when to take them. When a medication is prescribed the app user simply adds it to their account. They enter the name, dosage, frequency prescribed and the number of pills in the bottle. The app shows what medications are due by time of day. The user then clicks the taken button and the app will keep track of what medications were taken and the number of pills remaining. Clicking on the drug info icon opens up relevant patient information about the drug. We utilize an API provided by the FDA (https://open.fda.gov/about/) for the drug generic name and patient information.
The app is built using HTML, JavaScript, JQuery, CSS, Bootstrap, Node JS, Handlebars, MySQL and Sequelize.
This is a responsive web-based application that works on tablets, smart phones and other mobile devices without the need for a download.
Drug data is provided by the FDA and is delivered to the front end by querying the API at the time of user request so is the most current drug information.
MySQL and Sequelize were used to build the database that houses the user data and the medication list.
